## ⚡ Calculadora de Consumo de Energia

### 📌 Nome e objetivo do sistema
Este projeto, **Calculadora de Consumo de Energia**, tem como objetivo estimar o consumo mensal de energia elétrica de um aparelho, ajudando a entender melhor o gasto energético no dia a dia.

---

### 🐍 Linguagem utilizada
O sistema foi desenvolvido em **Python**, utilizando conceitos básicos de entrada de dados, cálculo e exibição de resultados.

---
